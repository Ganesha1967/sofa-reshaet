# Сайт репетитора "Софа всё решит" - Математика и Физика

Современный и адаптивный лендинг для профессионального репетитора по математике и физике

🔗 **Демо:** [sofa-vse-reshaet](https://ganesha1967.github.io/sofa-vse-reshaet)

## 🛠 Технологический стек

- **HTML5** - семантическая разметка
- **Tailwind CSS** - utility-first стилизация и адаптивность
- **JavaScript** - интерактивность интерфейса

## Структура проекта

```bash
├── assets/
│   ├── css/
│   │   └── style.css          # Скомпилированные Tailwind-стили
│   └── images/
│       ├── main-cat.jpeg
│       └── sofa.jpg
├── index.html                 # Главная страница
├── package.json               # Конфигурация проекта
├── package-lock.json
└── LICENSE
```

## Локальный запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/Ganesha1967/sofa-vse-reshaet.git
cd sofa-vse-reshaet
```

### 2. Установка зависимостей

```bash
npm install
```

## Для контрибьюторов

Проект автоматически публикуется через **GitHub Pages** при пуше в ветку `main`.  
Для корректной работы на GitHub Pages используйте относительные пути к ресурсам внутри `index.html`.

Всю работу по исправлению чего-либо делайте в отдельных ветках и отправляйте **pull request** с подробным описанием того, что было изменено или добавлено.
